# mini-tools

A collection of small, focused command-line tools for personal use.

## Tools

| Tool | Description |
| --- | --- |
| [plog](plog/) | Interactive terminal progress logger — appends timestamped entries to a Markdown and JSONL file |

Each tool lives in its own directory with its own README covering setup and usage.

## Philosophy

- **Single-purpose** — each tool solves one specific problem
- **Minimal dependencies** — prefer the standard library where possible
- **Easy to read** — small enough that you can understand the whole tool in a few minutes
